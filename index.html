sobre mim

Meu nome é Leandro França, profissional de frontend apaixonado por programação. Com habilidades em HTML, CSS, JavaScript e outras linguagens de programação, estou sempre pronto para enfrentar desafios e encontrar soluções criativas. Tenho orgulho de dizer que minha paixão por programação tem sido um diferencial em cada projeto que participei. Meu objetivo é criar experiências de usuário incríveis, que sejam não apenas bonitas, mas também funcionais e intuitivas. Ao navegar pelo meu site, você encontrará exemplos dos meus projetos mais recentes, que mostram como minha paixão se reflete em minha abordagem ao desenvolvimento de interfaces para a web. Se você tem um projeto desafiador que precisa de uma solução criativa, estou pronto para ajudá-lo(a). Vamos conversar e encontrar a melhor solução para as suas necessidades.
